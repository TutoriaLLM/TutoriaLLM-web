# Contributing

This app is developed as open-source, so anyone can contribute. We would appreciate your contributions in the following ways:

<LinkCard
  title="GitHub Repository"
  description="Check out the GitHub repository for TutoriaLLM."
  href="https://github.com/TutoriaLLM/TutoriaLLM"
/>

<Aside type="caution" title="Caution!">
  As of August 2024, this app is still in the early stages of development and
  has many unstable areas. Additionally, the methods for contributing are not
  yet well-established. Please understand this.
</Aside>

## Reporting Bugs
